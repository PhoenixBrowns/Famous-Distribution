# Famous Distribution (Official Company Website)

## Project Overview
This repository contains the official web presence for Famous Distribution, a distribution company in Keng Tung founded by my father. 

### Note for Admissions: 
This project is currently in Phase 1 (Product Landing Page). More developments will be integrated in near future.

## Core Features

**CSS Flexbox Layout**: A staff directory and testimonial section using CSS Flexbox to maintain perfect alignment across cards.

**Market-Standard Coding**: Uses good coding practices and standard coding procedures.

**Strategic Branding**: Uses a consistent color palette (navy, gold, and deep red) that aligns with the "Famous Distribution" identity and the staff uniforms.

**Persistent Navigation**: A fixed header that ensures the company brand and primary links are always available to the user.

**Semantic Data**: Implementation of the semantically accessible tags such as `<section>` for dividers, `<hr>` for line breaks, `<address>` for contact information and `<blockquote>` tags for retailer reviews.

**"Under-Development" Page**: A page coded to show that the requrested URL is under construction.
